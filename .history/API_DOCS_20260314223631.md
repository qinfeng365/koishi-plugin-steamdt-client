# SteamDT 异动数据 API 文档

## 概述

这是一个 RESTful API 服务，用于爬取和处理 SteamDT CS2 饰品异动数据。

## 快速开始

### 启动服务

```bash
python api_server.py
```

服务将在 `http://localhost:8000` 启动

### 访问 API 文档

- **Swagger UI**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc

## API 端点

### 1. 获取服务信息

**请求:**
```
GET /
```

**响应:**
```json
{
  "name": "SteamDT 异动数据 API",
  "version": "1.0.0",
  "status": "运行中",
  "endpoints": { ... }
}
```

### 2. 获取爬虫状态

**请求:**
```
GET /api/status
```

**响应:**
```json
{
  "is_running": false,
  "progress": 100,
  "total": 120,
  "message": "完成！共爬取 120 条数据",
  "timestamp": "2026-03-14T22:36:00.000000"
}
```

### 3. 开始爬取数据

**请求:**
```
POST /api/scrape?scrolls=10
```

**参数:**
- `scrolls` (int): 滑动次数，范围 1-50（默认 10）

**响应:**
```json
{
  "status": "已启动",
  "scrolls": 10,
  "message": "爬虫已启动，请稍候...",
  "timestamp": "2026-03-14T22:36:00.000000"
}
```

### 4. 获取爬取的数据

**请求:**
```
GET /api/data?limit=50
```

**参数:**
- `limit` (int): 限制返回条数（可选）

**响应:**
```json
{
  "total": 120,
  "data": [
    {
      "timestamp": "今天 22:04",
      "name": "MAC-10 | 绝界之行",
      "old_price": "74.89",
      "new_price": "75",
      "price_change": "74.89 >> 75",
      "price_trend": "上涨",
      "price_change_percent": "4.16",
      "image_url": "https://...",
      "full_content": "...",
      "scraped_at": "2026-03-14T22:08:15.047942"
    }
  ],
  "timestamp": "2026-03-14T22:36:00.000000"
}
```

### 5. 下载 JSON 数据

**请求:**
```
GET /api/data/json
```

**响应:** 返回 JSON 文件下载

### 6. 下载 CSV 数据

**请求:**
```
GET /api/data/csv
```

**响应:** 返回 CSV 文件下载

### 7. 下载表格图片

**请求:**
```
GET /api/table
```

**响应:** 返回 PNG 图片文件下载

### 8. 获取数据统计

**请求:**
```
GET /api/stats
```

**响应:**
```json
{
  "total": 120,

  "down_count": 40,

  "max_up": {
    "percent": 7.25,
```

### 9. 打开数据查看器

**请求:**
```
GET /viewer
```

**响应:** 返回 HTML 查看器页面

### 10. 健康检查

**请求:**
```
GET /health
```

**响应:**
```json
{
  "status": "healthy",
  "timestamp": "2026-03-14T22:36:00.000000"
}
```

## 使用示例

### Python 客户端

```python
import requests
import json

# API 基础 URL
BASE_URL = "http://localhost:8000"

# 1. 开始爬取数据
response = requests.post(f"{BASE_URL}/api/scrape?scrolls=10")
print(response.json())

# 2. 获取爬虫状态
response = requests.get(f"{BASE_URL}/api/status")
print(response.json())

# 3. 获取数据
response = requests.get(f"{BASE_URL}/api/data?limit=50")
data = response.json()
print(f"总条数: {data['total']}")

# 4. 获取统计信息
response = requests.get(f"{BASE_URL}/api/stats")
stats = response.json()
print(f"平均涨幅: {stats['avg_change']}%")

# 5. 下载 JSON 数据
response = requests.get(f"{BASE_URL}/api/data/json")
with open('data.json', 'wb') as f:
    f.write(response.content)

# 6. 下载表格图片
response = requests.get(f"{BASE_URL}/api/table")
with open('table.png', 'wb') as f:
    f.write(response.content)
```

## 工作流示例

### 完整的数据获取流程

```python
import requests
import time

BASE_URL = "http://localhost:8000"

# 1. 启动爬虫
print("启动爬虫...")
response = requests.post(f"{BASE_URL}/api/scrape?scrolls=10")
print(response.json())

# 2. 等待爬虫完成
print("等待爬虫完成...")
while True:
    response = requests.get(f"{BASE_URL}/api/status")
    status = response.json()
    print(f"进度: {status['progress']}% - {status['message']}")
    
    if not status['is_running']:
        break
    
    time.sleep(2)

# 3. 获取统计信息
print("\n获取统计信息...")
response = requests.get(f"{BASE_URL}/api/stats")
stats = response.json()
print(f"总条数: {stats['total']}")
print(f"上涨: {stats['up_count']}")
print(f"下跌: {stats['down_count']}")
print(f"平均涨幅: {stats['avg_change']}%")

# 4. 下载数据
print("\n下载数据...")
response = requests.get(f"{BASE_URL}/api/data/json")
with open('data.json', 'wb') as f:
    f.write(response.content)

response = requests.get(f"{BASE_URL}/api/table")
with open('table.png', 'wb') as f:
    f.write(response.content)

print("完成！")
```

## 错误处理

### 常见错误

| 状态码 | 错误信息 | 解决方案 |
|--------|---------|---------|
| 400 | 爬虫已在运行中 | 等待爬虫完成后再启动 |
| 400 | 滑动次数必须在 1-50 之间 | 调整 scrolls 参数 |
| 404 | 数据文件不存在 | 先爬取数据 |
| 500 | 内部服务器错误 | 查看服务器日志 |

## 性能指标

| 操作 | 耗时 | 数据量 |
|------|------|--------|
| 爬取（5 次滑动） | 30-40 秒 | 120 条 |
| 爬取（10 次滑动） | 60-80 秒 | 200+ 条 |
| 生成表格图片 | 5-10 秒 | 全部数据 |
| 获取数据 API | < 1 秒 | 全部数据 |

## 部署建议

### 生产环境

```bash
# 使用 Gunicorn + Uvicorn
pip install gunicorn

gunicorn api_server:app \
  --workers 4 \
  --worker-class uvicorn.workers.UvicornWorker \
  --bind 0.0.0.0:8000
```

### Docker 部署

```dockerfile
FROM python:3.11

WORKDIR /app

COPY requirements.txt .
RUN pip install -r requirements.txt

COPY . .

CMD ["python", "api_server.py"]
```
