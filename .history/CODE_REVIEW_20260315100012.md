### 3. **线程安全问题** [`api_server.py:106-122`](api_server.py:106)
**严重级别**: 🔴 严重

**问题描述**:
在后台线程中创建新的事件循环，可能导致事件循环冲突：

```python
loop = asyncio.new_event_loop()
asyncio.set_event_loop(loop)
scraper = SteamDTScraper(max_scrolls=task['scrolls'])
data = loop.run_until_complete(scraper.run())
loop.close()
```

**风险**:
- 多线程环境下事件循环管理不当
- 可能导致 "RuntimeError: Event loop is closed"
- 资源泄漏

**建议修复**:
使用 `asyncio.run()` 或在线程池中运行：
```python
import concurrent.futures

def run_scraper_sync(scrolls):
    return asyncio.run(SteamDTScraper(max_scrolls=scrolls).run())

# 在线程池中执行
with concurrent.futures.ThreadPoolExecutor() as executor:
    data = executor.submit(run_scraper_sync, task['scrolls']).result()
```

### 4. **CORS 配置过于宽泛** [`api_server.py:32-38`](api_server.py:32)
**严重级别**: 🔴 严重

**问题代码**:
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

**安全风险**:
- 允许任何来源的跨域请求
- 与 `allow_credentials=True` 组合使用违反安全最佳实践
- 容易被恶意网站利用

**建议修复**:
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["https://yourdomain.com", "https://app.yourdomain.com"],
    allow_credentials=False,  # 如果需要凭证，不要使用通配符
    allow_methods=["GET", "POST"],
    allow_headers=["Content-Type"],
)
```

### 7. **正则表达式过于复杂且缺少验证** [`steamdt_scraper_final.py:96-112`](steamdt_scraper_final.py:96)
**严重级别**: 🟠 主要

**问题代码**:
```python
price_match = re.search(r'¥\s*([\d.]+)\s*>>\s*([\d.]+)', all_text)
old_price = price_match.group(1) if price_match else ''
```

**问题**:
- 正则表达式复杂，难以维护
- 缺少数据验证（例如价格是否为有效数字）
- 提取失败时返回空字符串，后续处理可能出错

**建议修复**:
```python
def extract_price(text):
    """提取价格信息"""
    match = re.search(r'¥\s*([\d.]+)\s*>>\s*([\d.]+)', text)
    if not match:
        return None, None
    
    try:
        old_price = float(match.group(1))
        new_price = float(match.group(2))
        return old_price, new_price
    except ValueError:
        logger.warning(f"价格转换失败: {match.groups()}")
        return None, None
```

### 8. **内存泄漏风险** [`api_server_async.py:40-41`](api_server_async.py:40)
**严重级别**: 🟠 主要

**问题代码**:
```python
tasks: Dict[str, dict] = {}
websocket_connections: List[WebSocket] = []
```

**问题**:
- 全局字典无限增长，已完成的任务永不删除
- WebSocket 连接列表可能包含已断开的连接
- 长期运行会导致内存溢出

**建议修复**:
```python
from collections import OrderedDict
from datetime import datetime, timedelta

class TaskManager:
    MAX_TASKS = 1000
    TASK_RETENTION_HOURS = 24
    
    @staticmethod
    def cleanup_old_tasks():
        """清理过期任务"""
        cutoff_time = datetime.now() - timedelta(hours=TaskManager.TASK_RETENTION_HOURS)
        expired = [
            task_id for task_id, task in tasks.items()
            if task.get("completed_at") and 
            datetime.fromisoformat(task["completed_at"]) < cutoff_time
        ]
        for task_id in expired:
            del tasks[task_id]
```

class ChangeItem(BaseModel):
    timestamp: str
    name: str
    old_price: str
    new_price: str
    price_trend: str
    # ...

def load_and_validate_data(json_file: str) -> List[ChangeItem]:
    """加载并验证数据"""
