# SteamDT 异动数据 API 文档 - 实时版本

## 概述

这是一个 RESTful API 服务，**每次调用时实时爬取数据**并返回表格图片、JSON 或 CSV 格式的数据。

## 快速开始

### 启动服务

```bash
python api_server.py
```

服务将在 `http://localhost:8000` 启动

### 访问 API 文档

- **Swagger UI**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc

## API 端点

### 1. 获取服务信息

**请求:**
```
GET /
```

**响应:**
```json
{
  "name": "SteamDT 异动数据 API - 实时版本",
  "version": "2.0.0",
  "status": "运行中",
  "description": "每次调用时实时爬取数据并返回表格图片",
  "endpoints": { ... }
}
```

### 2. 爬取数据并返回表格图片 ⭐

**请求:**
```
GET /api/scrape/table?scrolls=10
```

**参数:**
- `scrolls` (int): 滑动次数，范围 1-50（默认 10）

**响应:** 返回 PNG 表格图片文件

**示例:**
```bash
curl "http://localhost:8000/api/scrape/table?scrolls=10" -o table.png
```

### 3. 爬取数据并返回 JSON

**请求:**
```
GET /api/scrape/json?scrolls=10&limit=50
```

**参数:**
- `scrolls` (int): 滑动次数，范围 1-50（默认 10）
- `limit` (int): 限制返回条数（可选）

**响应:**
```json
{
  "total": 120,
  "scrolls": 10,
  "data": [
    {
      "timestamp": "今天 22:04",
      "name": "MAC-10 | 绝界之行",
      "old_price": "74.89",
      "new_price": "75",
      "price_change": "74.89 >> 75",
      "price_trend": "上涨",
      "price_change_percent": "4.16",
      "image_url": "https://...",
      "full_content": "...",
      "scraped_at": "2026-03-14T22:08:15.047942"
    }
  ],
  "timestamp": "2026-03-14T22:36:00.000000"
}
```

### 4. 爬取数据并返回 CSV

**请求:**
```
GET /api/scrape/csv?scrolls=10
```

**参数:**
- `scrolls` (int): 滑动次数，范围 1-50（默认 10）

**响应:** 返回 CSV 文件下载

**示例:**
```bash
curl "http://localhost:8000/api/scrape/csv?scrolls=10" -o data.csv
```

### 5. 爬取数据并返回统计信息

**请求:**
```
GET /api/scrape/stats?scrolls=10
```

**参数:**
- `scrolls` (int): 滑动次数，范围 1-50（默认 10）

**响应:**
```json
{
  "total": 120,
  "scrolls": 10,
  "up_count": 80,
  "down_count": 40,
  "avg_change": 2.45,
  "max_up": {
    "percent": 7.25,
    "content": "摩托手套（★） | 小心烟雾弹..."
  },
  "max_down": {
    "percent": -2.96,
    "content": "沙漠之鹰 | 午夜凶匪..."
  },
  "timestamp": "2026-03-14T22:36:00.000000"
}
```

### 6. 健康检查

**请求:**
```
GET /health
```

**响应:**
```json
{
```

## 使用示例

### Python 客户端

```python
import requests
from datetime import datetime

BASE_URL = "http://localhost:8000"

# 1. 获取表格图片
print("获取表格图片...")
response = requests.get(f"{BASE_URL}/api/scrape/table?scrolls=10")
with open(f'table_{datetime.now().strftime("%Y%m%d_%H%M%S")}.png', 'wb') as f:
    f.write(response.content)
print("✓ 表格图片已保存")

# 2. 获取 JSON 数据
print("\n获取 JSON 数据...")
response = requests.get(f"{BASE_URL}/api/scrape/json?scrolls=10&limit=50")
data = response.json()
print(f"✓ 获取 {data['total']} 条数据")

# 3. 获取 CSV 数据
print("\n获取 CSV 数据...")
response = requests.get(f"{BASE_URL}/api/scrape/csv?scrolls=10")
with open(f'data_{datetime.now().strftime("%Y%m%d_%H%M%S")}.csv', 'wb') as f:
    f.write(response.content)
print("✓ CSV 数据已保存")

# 4. 获取统计信息
print("\n获取统计信息...")
response = requests.get(f"{BASE_URL}/api/scrape/stats?scrolls=10")
stats = response.json()
print(f"✓ 总条数: {stats['total']}")
print(f"  上涨: {stats['up_count']}")
print(f"  下跌: {stats['down_count']}")
print(f"  平均涨幅: {stats['avg_change']}%")
```

### JavaScript 示例

```javascript
const BASE_URL = 'http://localhost:8000';

// 获取表格图片
async function getTableImage() {
  const response = await fetch(`${BASE_URL}/api/scrape/table?scrolls=10`);
  const blob = await response.blob();
  
  // 创建下载链接
  const url = window.URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = `table_${new Date().toISOString().split('T')[0]}.png`;
  a.click();
}

// 获取 JSON 数据
async function getJsonData() {
  const response = await fetch(`${BASE_URL}/api/scrape/json?scrolls=10`);
  const data = await response.json();
  console.log(`获取 ${data.total} 条数据`);
  return data;
}

// 获取统计信息
async function getStats() {
  const response = await fetch(`${BASE_URL}/api/scrape/stats?scrolls=10`);
  const stats = await response.json();
  console.log(`平均涨幅: ${stats.avg_change}%`);
  return stats;
}
```

## 工作流示例

### 完整的数据获取流程

```python
import requests
import json
from datetime import datetime

BASE_URL = "http://localhost:8000"

print("=" * 60)
print("SteamDT 异动数据获取流程")
print("=" * 60)

# 1. 获取表格图片
print("\n[1/3] 获取表格图片...")
response = requests.get(f"{BASE_URL}/api/scrape/table?scrolls=10")
if response.status_code == 200:
    filename = f'table_{datetime.now().strftime("%Y%m%d_%H%M%S")}.png'
    with open(filename, 'wb') as f:
        f.write(response.content)
    print(f"✓ 表格图片已保存: {filename}")
else:
    print(f"✗ 获取失败: {response.status_code}")

# 2. 获取统计信息
print("\n[2/3] 获取统计信息...")
response = requests.get(f"{BASE_URL}/api/scrape/stats?scrolls=10")
if response.status_code == 200:
    stats = response.json()
    print(f"✓ 数据统计:")
    print(f"  总条数: {stats['total']}")
    print(f"  上涨: {stats['up_count']}")
    print(f"  下跌: {stats['down_count']}")
    print(f"  平均涨幅: {stats['avg_change']}%")
    if stats['max_up']:
        print(f"  最大涨幅: {stats['max_up']['percent']}%")
    if stats['max_down']:
        print(f"  最大跌幅: {stats['max_down']['percent']}%")
else:
    print(f"✗ 获取失败: {response.status_code}")

# 3. 获取 JSON 数据
print("\n[3/3] 获取 JSON 数据...")
response = requests.get(f"{BASE_URL}/api/scrape/json?scrolls=10&limit=20")
if response.status_code == 200:
    data = response.json()
    filename = f'data_{datetime.now().strftime("%Y%m%d_%H%M%S")}.json'
    with open(filename, 'w', encoding='utf-8') as f:
        json.dump(data, f, ensure_ascii=False, indent=2)
    print(f"✓ JSON 数据已保存: {filename}")
else:
    print(f"✗ 获取失败: {response.status_code}")

print("\n" + "=" * 60)
print("✓ 所有操作完成！")
print("=" * 60)
```

## 错误处理

### 常见错误

| 状态码 | 错误信息 | 解决方案 |
|--------|---------|---------|
| 400 | 滑动次数必须在 1-50 之间 | 调整 scrolls 参数 |
| 500 | 爬取数据失败 | 检查网络连接 |
| 500 | 生成表格图片失败 | 检查依赖库是否安装 |

## 性能指标

| 操作 | 耗时 | 数据量 |
|------|------|--------|
| 爬取（5 次滑动） | 30-40 秒 | 120 条 |
| 爬取（10 次滑动） | 60-80 秒 | 200+ 条 |
| 生成表格图片 | 5-10 秒 | 全部数据 |
| 返回 JSON | < 1 秒 | 全部数据 |

## 部署建议

### 生产环境

```bash
# 使用 Gunicorn + Uvicorn
pip install gunicorn

gunicorn api_server:app \
  --workers 4 \
  --worker-class uvicorn.workers.UvicornWorker \
  --bind 0.0.0.0:8000
```

### Docker 部署

```dockerfile
FROM python:3.11

WORKDIR /app

COPY requirements.txt .
RUN pip install -r requirements.txt
RUN playwright install chromium

COPY . .

CMD ["python", "api_server.py"]
```
