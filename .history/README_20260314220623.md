# SteamDT 异动数据爬虫

这是一个用于爬取 [SteamDT](https://steamdt.com/changes) CS2 饰品异动数据的 Python 爬虫。

## 功能特性

- 爬取 CS2 饰品实时异动数据
- 提取商品名称、价格变化、时间戳等信息
- 支持导出为 JSON 和 CSV 格式
- 使用 Playwright 处理动态加载的页面内容

## 安装依赖

```bash
pip install -r requirements.txt
playwright install chromium
```

## 使用方法

### 方式 1：运行最终版爬虫（推荐）

```bash
python steamdt_scraper_final.py
```

这将爬取数据并保存到：
- `steamdt_changes.json` - JSON 格式
- `steamdt_changes.csv` - CSV 格式

### 方式 2：在 Python 代码中使用

```python
import asyncio
from steamdt_scraper_final import SteamDTScraper

async def main():
    scraper = SteamDTScraper(output_dir='./data')
    data = await scraper.run()
    print(f"爬取了 {len(data)} 条数据")

asyncio.run(main())
```

## 数据格式

### JSON 格式示例

```json
[
  {
    "timestamp": "今天 22:02",
    "name": "AK-47 | 狩猎网格",
    "old_price": "99.86",
    "new_price": "101.95",
    "price_change": "99.86 >> 101.95",
    "image_url": "https://img.zbt.com/...",
    "full_content": "今天 22:02BUFFAK-47 | 狩猎网格 (崭新出厂)在售价¥ 99.86 >> 101.95...",
    "scraped_at": "2026-03-14T22:04:24.439621"
  }
]
```

### CSV 格式

包含以下列：
- `timestamp` - 异动时间
- `name` - 商品名称
- `old_price` - 旧价格
- `new_price` - 新价格
- `price_change` - 价格变化
- `image_url` - 商品图片 URL
- `full_content` - 完整内容
- `scraped_at` - 爬取时间

## 注意事项

1. 首次运行需要下载 Chromium 浏览器（约 150MB）
2. 爬虫会等待页面完全加载，可能需要 10-15 秒
3. 请遵守网站的 robots.txt 和使用条款
4. 建议在爬取大量数据时添加延迟以避免对服务器造成压力

## 故障排除

### 问题：找不到 Chromium

**解决方案：**
```bash
playwright install chromium
```

### 问题：页面加载超时

**解决方案：** 检查网络连接，或增加超时时间（修改代码中的 `timeout=30000`）

### 问题：编码错误

**解决方案：** 确保使用 UTF-8 编码打开文件
